## 招聘

### 掘金 Node.js 开发

基本要求：
- 3 年以上工作经验（或水平符合）
- 熟练掌握 JavaScript 语法及模块化编程
- 熟练掌握 Express.js
- 熟练掌握 Git 及 Git 部署
- 了解基础前端支持
- 了解 Grunt, Gulp, Webpack
- 了解数据库（SQl，NoSQL）

加分项：
- 会使用 ES6 语法
- 熟练掌握 Webpack
- 有基于 LeanCloud 开发经验
- 对前端框架 Vue.js 或 React.js 有了解

工作内容：
- 掘金网站后台开发
- 掘金后端脚本及接口
- 掘金插件开发

薪资： **20K - 35K / 月**

- 互联网重度使用者

加分项：
